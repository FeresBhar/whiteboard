# Collaborative Whiteboard

A real-time collaborative drawing application where multiple users can draw together on shared pages.

## Features

- Real-time drawing synchronization
- Multiple pages with thumbnails
- Zoom in/out
- Undo and clear functions
- Color picker and brush thickness
- Touch support for mobile devices

## Setup

1. Update `canvas.js` line 20 with your backend URL
2. Deploy to GitHub Pages or serve locally

## Usage

- Draw with mouse or touch
- Use toolbar to change colors and thickness
- Navigate between pages using sidebar
- Add new pages with the green button
- Delete pages by hovering and clicking Ã—

## Rooms

Add `?room=roomname` to URL to create separate rooms:
- `?room=design`
- `?room=meeting`
- `?room=team1`
